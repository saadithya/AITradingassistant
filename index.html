Privacy Policy for AI Trading Monitor

Effective Date: August 16, 2025

AI Trading Monitor (“the extension”) respects your privacy. This Privacy Policy explains what information the extension collects, how it is used, and your rights.

1. Information We Collect

  • API Keys (User Provided): If you choose to use AI-powered analysis, you may enter an API key for OpenAI or Anthropic (Claude). These keys are stored locally in your browser using Chrome’s storage.sync and are only transmitted securely to the selected provider for authentication. They are never shared with third parties.
  • Chart Data (From Supported Sites): The extension extracts publicly available market data (price, volume, symbol, historical values) from supported sites (TradingView, Binance, Yahoo Finance) to provide alerts.
  • Usage Statistics (Local Only): Daily counts of alerts triggered and API requests are stored locally in your browser for functionality.

2. Information We Do Not Collect

The extension does not collect:

  • Personally identifiable information (name, email, address, etc.)
  • Financial or payment details (payments are handled externally by Gumroad)
  • Browsing history, keystrokes, or unrelated page content
  • Location data

3. How We Use Information

  • To provide AI-powered analysis and alerts based on user-defined conditions.
  • To display notifications and play sounds when alerts are triggered.
  • To verify premium license status (via Gumroad API).

4. Data Sharing

  • No user data is sold, rented, or shared with third parties.
  • API keys are only used to connect directly to OpenAI/Anthropic services.
  • Licensing verification is handled securely with Gumroad.

5. Security

We use Chrome’s extension storage APIs to keep data safe on your device. Communication with AI providers and Gumroad is encrypted via HTTPS.

6. User Control

  • Remove or reset your API keys at any time in the extension settings.
  • Uninstall the extension to delete all stored data.

7. Changes

We may update this Privacy Policy as the extension evolves. Updates will be posted on this page.
